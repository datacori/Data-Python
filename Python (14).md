### Numerical Computing Library (NumPy)

- **NumPy (Numerical Python)** is a core library for numerical and scientific computing in Python  
- A foundational Python library for mathematical and numerical analysis  
- Provides efficient support for multi-dimensional arrays and a wide range of functions required for scientific calculations

### Example


# Data Visualization Libraries in Python

This section summarizes commonly used Python libraries for data visualization, focusing on **Matplotlib** and **Seaborn**.

---

## Matplotlib

**Matplotlib** is a core Python library used for visualizing data through charts and plots.

### Key Features
- Enables data visualization using various chart and plot types
- Provides full control over graph creation and customization
- Widely used as the foundation for many other visualization libraries

### Installation
```bash
pip install matplotlib
```

### Example

```python
import matplotlib.pyplot as plt
```

## Seaborn

**Seaborn** is a high-level data visualization library built on the top of Matplotlib.

### Key Features
- Based on Matplotlib with improved default styles
- Designed for creating statistical visualizations
- Offers a simpler and more intuitive interface for complex plots

### Installation
```bash
pip install seaborn
```

### Example
```python
import seaborn as sns
```

# Data Collection & Machine Learning Libraries in Python

## BeautifulSoup

**BeautifulSoup** is a Python library used for extracting and parsing data from HTML and XML documents.

### Key Features
- Simplifies web data extraction from HTML and XML files
- Helps handle malformed or poorly structured HTML
- Commonly used to clean and process web data before analysis

### Installation
```bash
pip install beautifulsoup4
```

### Example
```python
from bs4 import BeautifulSoup
```

## Python Development Environments

This section introduces commonly used **Python development environments** for data analysis and experimentation.

---

## Jupyter Notebook

**Jupyter Notebook** is an open-source web application widely used for data analysis and experimentation.

### Key Features
- Open-source, web-based interactive computing environment
- Allows multiple code cells to be written and executed independently
- Combines code, output, and documentation in a single notebook
- Enables users to run selected parts of code and immediately review results

---

## Google Colaboratory (Colab)

**Google Colaboratory (Colab)** is a cloud-based Jupyter Notebook environment provided by Google.

### Key Features
- Integrates Google Drive with Jupyter Notebook
- Requires no local setup and runs entirely in the browser
- Ideal for collaboration and sharing notebooks
- Commonly used for open-source and educational projects

---

## Summary

- **Jupyter Notebook** is ideal for local data exploration and iterative analysis  
- **Google Colab** offers a cloud-based alternative with easy setup and collaboration  
- Both environments are widely used in data science and machine learning workflows
